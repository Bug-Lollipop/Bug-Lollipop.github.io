前端实现图片上传

移动端

移动端的H5通过 JsBridge 上传图片，返回base64，然后我们需要将base64 转换成Blob 二进制文件，再发送给后端，代码如下：


export const convertBase64UrlToBlob = dataURI => {
  const arr = dataURI.split(',')
  const mime = arr[0].match(/:(.*?);/)[1]
  
  //去掉base64中的换行符，webkit会自动去除，但是IOS9以及IOS8中不会自动去除，导致转换出错
  const bstr = atob(arr[1].replace(/\s/g, ''))
  let n = bstr.length
  const u8arr = new Uint8Array(n)
  while (n--) {
    u8arr[n] = bstr.charCodeAt(n)
  }
  return new Blob([u8arr], { type: mime })
}

margin 合并

什么是margin 合并

块级元素的上下边距有时会合并成单个边距，这种现象就是 margin 合并。
该现象只发生在和当前文档流方向垂直的方向上。由于默认文档流是水平流，所以发生margin 合并的是垂直方向。

margin 合并的3种场景

相邻元素margin 合并

这个比较常见：

p {
  margin: 1em 0;
}
<p>第一行</p>
<p>第二行</p>

实际效果是 第一行和第二行的间距还是 1em。

【译】如何逃离 async / await 地狱

async/ await 使我们从回调地狱中解脱出来。但是人们又开始被它虐待 - 进入了 async/ await 地狱

在这片文章中，我将会尝试去解释 async/await 地狱是什么，同时分享一些解决方案。

什么是 async/await 地狱

当 Javascript 异步编程的时候时，人们总是使用很多 await 语句，并在函数调用之前总是等待它。这会导致性能问题，因为一个语句多次调用并不依赖前一个语句 - 但是你仍然必须等待前一个语句的完成。

一个 async/ await 地狱的例子

尝试去写一个去实现购买披萨和饮料的功能。这个代码可能是这个样子的：

(async () => {
  const pizzaData = await getPizzaData()    // async call
  const drinkData = await getDrinkData()    // async call
  const chosenPizza = choosePizza()    // sync call
  const chosenDrink = chooseDrink()    // sync call
  await addPizzaToCart(chosenPizza)    // async call
  await addDrinkToCart(chosenDrink)    // async call
  orderItems()    // async call
})()

font-size: 0 与文本隐藏

chrome 中文浏览器有个12px 的字号限制，就是文字的font-size 计算值不能小于12px。

所以设置了小于12px 的字体会被当做12px处理，但是有个例外，那就是0，也就是说，如果font-size: 0 的字号表现就是0，那么文本就可以直接被隐藏掉，并且只能是 font-size: 0，哪怕设置成 fong-size: 0.00001px，都会被当做12px处理。

因此希望隐藏logo对应元素内的文字，除了text-indent 缩进隐藏，还可以用：

.logo {
  font-size: 0;
}

【译】display: content 是如何工作的

原文： https://bitsofco.de/how-display-contents-works/

就像经常提到的那样，每个元素在文档流中都是一个长方形的盒子，简单解释是：这个“长方形的盒子” 由两部分组成。第一部分是一个真实的盒子，由border， padding 和 margin 的区域组成。接下来一部分是盒子的内容区域。

CSS display 属性可以让我们去控制如何在这个盒子中及它的子元素在页面上的排列，我们可以用inline 属性使这个盒子和兄弟元素在同一行，也可以用 table 属性使其呈现表格的状态，我们甚至可以用 absolute 属性，在不同事物 Z 轴上堆叠这个盒子。
display 属性有两个可能值可以控制并标记指定的元素生成方框。none 值 会使盒子和内容都不在页面中绘制。另一方面，content 值是一个新的特性，盒子中的文本将会被正常绘制，而围绕在盒子中的框并不会生成。

229. Maximum Subarray

Find the contiguous subarray within an array (containing at least one number) which has the largest sum.

For example, given the array [-2,1,-3,4,-1,2,1,-5,4],
the contiguous subarray [4,-1,2,1] has the largest sum = 6.

/**
 * @param {number[]} nums
 * @return {number}
 */
var maxSubArray = function(nums) {
    var len = nums.length
    var maxNum = nums[0]
    var curNum = 0
    var endSum = nums[len-1]
    if (len === 1) maxNum = nums[0]
    else {
        for (var i = 0; i<len; i++) {
        curNum = nums[i]
        maxNum = Math.max(maxNum, curNum)
        for (var j = i+1; j< len; j++) {

        }

     }
   }
   return maxNum
};
