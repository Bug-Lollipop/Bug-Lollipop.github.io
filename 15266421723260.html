优雅的增加点击区域的大小

移动端我们点击屏幕上很小的箭头时候，经常会很费经，要不就点不到，要不就误点到其他的地方，正常我们都是按照这个图标的原始尺寸写的，例如：

.icon-arrow {
  width: 16px;
  height: 16px;
}

效果没问题，但是肯定体验不友好，因为尺寸仅仅16px，我们很容易点不中。
稳妥的方法是外部在嵌套一层标签，专门控制点击区域大小，如果对代码要求比较高，则可以使用padding 或透明的border增加元素的点击区域大小。

其中首推透明 border方法，原因是如果我们的图标是工具生成的，那么background-position 就是限定死的值，若用 padding 撑开间距，就会遇到定位不准的问题。但是若用透明的border 增加点击区域，就没这个问题，只要合并时留下足够的间距就可以了，例如：

.icon-arrow {
  width: 16px;
  height: 16px;
  border: 11px solid transparent;
}
