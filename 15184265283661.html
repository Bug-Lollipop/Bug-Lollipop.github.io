~~的应用—— 对象数组Array 根据对象object key的值进行排序

按位非运算符“~”

简单说按位取反运算符(~) 的 法则 是 向下取整 后 求负 减1 ( -(x + 1))，如果说 单次 使用 Not 是 求负减1 ，那连续使用 2次 Not 对于 整型 的数字 就是它本身(-(-(x+1) + 1))，那么对于浮点型就是向下取整，布尔型转为数字

因此，对于浮点数，~~value可以代替parseInt(value)，而且前者效率更高些

下面来看一些具体例子：

~~ null // 0
~~ undefined // 0
~~ NaN // 0
~~ {} // 0
~~ true // 1
~~ '' // 0
~~ 'string' // 0
~~ '1' // 1
~~ 1.2 // 1
~~ -1.2 // -1
~~ 1.6 // 1
~~ -1.6 // -1

今天有个需求，需要将一个对象数组进行排序，可以用~~ 去实现

比如下面的数组对象，需要根据timeLineOrder 进行排序：

timeLineList ：[
{
"time":1504184661000,
"desc":"提现申请成功",
"timeLineOrder":1
},
{
"time":1504184662000,
"desc":"提现失败",
"timeLineOrder":3
},
{
"time":1504184661000,
"desc":"银行处理中",
"timeLineOrder":2
},
{
"time":1504184662000,
"desc":"退款失败",
"timeLineOrder":4
}
]

function keysrt(key,desc) {
  return (a,b) => {
    return desc ? ~~(a[key] < b[key]) : ~~(a[key] > b[key]);
  }
}

使用：

this.timeLineList = timeLineList.sort(this.keysrt('timeLineOrder', false))
