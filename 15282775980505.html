多种方式判断Javascript 对象中属性值为空

我们在做接口请求的时候，会出现在某种条件下某个参数是不需要传的，一般情况下，赋值时不需要的参数的属性值为null，undefined。这个时候需要过滤出属性值为null，undefined 的key，并删除。

下面总结了几种方式：

遍历该对象

const test = {
  test1 : null,
  test2 : 'somestring',
  test3 : 3,
}
function removeNullsInObject(obj) {
  for (var propName in obj) { 
    if (obj[propName] === null || obj[propName] === undefined) {
      delete obj[propName];
    }
  }
}
clean(removeNullsInObject);

如果担心此属性删除未运行对象的propttype 链，还可以：

function removeNullsInObject(obj) {
  const propNames = Object.getOwnPropertyNames(obj);
  for (var i = 0; i < propNames.length; i++) {
    let propName = propNames[i];
    if (obj[propName] === null || obj[propName] === undefined) {
      delete obj[propName];
    }
  }
}

使用lodash

lodash 中提供了 深度搜索对象中移除undefined值的方法：

var _ = require('lodash');

const test = {
  test1 : null,
  test2 : 'somestring',
  test3 : 3,
}

  return value !== null && value !== undefined;

使用JSON.stringify

将对象JSON.stringify之后，在JSON.parse组合将其重新转换为对象：

const test = {
    test1 : null,
    test2 : 'somestring',
    test3 : 3,
}
function removeNullsInObject(key, value) {
  if (value === null || value === undefined) {
    return undefined;
  }
return value;
}

console.log(JSON.parse(JSON.stringify(test, removeNullsInObject)));

看一下官方定义：

JSON.stringify(value[, replacer [, space]]) 的第二个参数replacer 可以是一个函数或者一个数组。
作为函数，它有两个参数，键(key)值(value)都会被序列化。
如果返回一个 Number, 转换成相应的字符串被添加入JSON字符串。
如果返回一个 String, 该字符串作为属性值被添加入JSON。
如果返回一个 Boolean, "true" 或者 "false"被作为属性值被添加入JSON字符串。
如果返回任何其他对象，该对象递归地序列化成JSON字符串，对每个属性调用replacer方法。除非该对象是一个函数，这种情况将不会被序列化成JSON字符串。
如果返回undefined，该属性值不会在JSON字符串中输出。
注意: 不能用replacer方法，从数组中移除值(values)，如若返回undefined或者一个函数，将会被null取代。

因此首先replacer 参数将值为null 或者undefined 通通转换成 undefined，然后其undefined 属性值不会在JSON字符串中输出的特性使得 值为null 的属性删除，然后再通过JSON.parse 还原成对象。

使用非空的键创建新对象

const test = {
    test1 : null,
    test2 : 'somestring',
    test3 : 3,
}
const removeNullsInObject = (obj) => {
  return Object.keys(obj).filter(key => obj[key]).reduce(
    (newObj, key) => {
      newObj[key] = obj[key]
      return newObj
    }, {}
  )
}

console.log(removeNullsInObject(test))
