currentColor - CSS 变量

在CSS 颜色（第三版)规范中，增加了很多颜色关键字，比如lightgoldenrodyellow等，不是很常用，但是我们还得到了一个特殊的关键字currentColor，他是从SVG那里借鉴来的。这个关键字并没有绑定到一个固定的颜色值，而是一直被解析为color。实际上，这个特性让它成为CSS中有史以来的第一个变量。

比如我们想让所有的水平分割线（所有的<hr>元素）自动与文本的颜色保持一致。
用currentColor我们可以这样写：

hr {
    height: .5em;
    backgorund: currentColor;
}
